# Getting Started

## Requirements

PHP: 8.1+ due to [enums](https://stitcher.io/blog/php-enums) and
[readonly class properties](https://stitcher.io/blog/php-81-readonly-properties).

## Installation

Install the package from [Packagist](https://packagist.org/packages/syndesi/webdav) by executing the
following command:

```bash
composer require syndesi/webdav
```
